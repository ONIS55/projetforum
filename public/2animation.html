Bienvenue !

Découvrez nos patterns d'animations avancées

1

Navigation avec fondu (fade-out)

1. Etat initial Page visible
2. Au clic "Aller" Fondu (opacity: 0)
3. Transition Navigation en cours
4. Nouvelle page Affichée

Description

Cette animation crée une transition douce lors de la navigation entre pages. Le contenu disparaît progressivement (fade-out), puis la nouvelle page apparaît. Parfait pour les transitions de page professionnelles.

2

Entrée fluide d'un message

1. Etat initial Pas de message
2. Inséré translateY(8px) + opacity: 0
3. Animation 220ms cubic-bezier
4. Visible translateY(0) + opacity: 1

Description

Cette animation fait glisser et apparaître les éléments progressivement. Idéale pour animer l'arrivée de nouveaux messages, commentaires, ou tout élément ajouté dynamiquement au DOM.

3

Lazy-loading des images

1. Images hors vue Placeholders
2. Dans viewport Observer détecte
3. Scroll bas Plus d'images chargées
4. Toutes visibles Chargement progressif

Description

Charge les images uniquement quand elles deviennent visibles dans le viewport. Réduit la bande passante et améliore les performances initiales de la page en chargeant progressivement.
